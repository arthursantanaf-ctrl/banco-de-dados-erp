# banco-de-dados-erp
# Banco de Dados ERP - Gestão de Projetos

Este repositório contém o script SQL para a criação do banco de dados do nosso projeto.

 Estrutura das Tabelas:
- clientes: Cadastro base de clientes.
- orçamentos: Orçamentos vinculados aos clientes (**FK**).
- faturas: Controle de cobranças.
- projetos: Gestão de projetos maiores.
- tarefas: Atividades ligadas aos projetos (**FK**).
- ordens_servico: OS técnicas.
- usuario: Dados de acesso ao sistema (Login).
- despesas: Controle financeiro interno.
